## Requires

- Linux
- nodejs and nodejs package manager (npm)

## Install

```sh
npm install
```

or if you like a global installation

```sh
npm install -g
```

Custom motd scripts can be placed into the `/etc/update-motd` folder.

## Run scripts

```sh
node [script].js
```
